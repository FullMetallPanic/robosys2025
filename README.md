-カードゲームなどで特定のカードが手札に含まれる確率を計算するツールです。

##実行例
-実行するとき、デッキの総数\引きたいカード\ドローする枚数という順番で数字を入力します。
'''
$echo -e 40\e4\e5 | ./syodou
0.4270707954918481
'''
